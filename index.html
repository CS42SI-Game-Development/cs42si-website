2D Game Development

Learn the fundamentals of game development using modern 2D game engines. CS42SI provides hands-on experience with industry-standard tools and techniques for creating engaging 2D games from concept to completion.

About the Course

In this hands-on, experiential course, students will design and develop 2D games using modern game engines. You'll learn how to use popular 2D game engines and frameworks, mastering the design principles and development pipeline for creating engaging 2D games. Students will work in teams to build complete games from concept to final product, learning industry-standard practices and tools. Enrollment is limited and by application only. Prerequisite: CS 106A or equivalent programming experience.

Join the Course

CS42SI is an open course - anyone is welcome to attend! No application required. Simply show up to the first class to enroll. This course is perfect for students interested in game development, interactive media, or software engineering.
